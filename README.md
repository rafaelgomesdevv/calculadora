# calculadora
Este é um projeto simples de uma calculadora implementada na linguagem C, que permite ao utilizador realizar operações matemáticas básicas, como soma, subtração, multiplicação e divisão. A calculadora apresenta um menu interativo onde o utilizador pode escolher a operação desejada, introduzir dois números e ver o resultado das operações em tempo real.

Funcionalidades:
Soma: Adiciona dois números fornecidos pelo utilizador.
Subtração: Subtrai o segundo número do primeiro.
Multiplicação: Multiplica os dois números.
Divisão: Divide o primeiro número pelo segundo, com verificação para evitar a divisão por zero.
Opção de sair: O utilizador pode sair do programa a qualquer momento.

O objetivo deste projeto é demonstrar a minha habilidade em programar em C.
É uma excelente forma de entender a lógica de controlo de fluxo, operações matemáticas e interação com o utilizador em programas simples de terminal.
